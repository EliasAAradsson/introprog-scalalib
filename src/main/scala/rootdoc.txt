This is the documentation of `luscala` - a simple-to-use library with Scala utilities used in beginner computer science teaching at Lund University.

== Package structure ==

- [[luscala.gui `luscala.gui`]] includes utilities for creating simple graphical user interfaces:
    - [[luscala.gui.SimpleCanvas `luscala.gui.SimpleCanvas`]] is a beginner-friendly class for creating drawing windows implemented using javafx via the  [[luscala.gui.Fx `luscala.gui.Fx`]] wrapper module.
